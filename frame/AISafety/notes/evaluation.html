    # ACC

    也称CA，未受攻击时，模型预测准确率。

    # ACAC

    1. 对错误类别的平均预测置信度。其定义为经过对抗攻击后，对于所有攻击成功对抗样本（FGSM、PGD、C&W），所有误分类类别的平均概率。
    2. 物理含义：将softmax层后的输出值当作分类置信度，计算在对抗攻击成功的分类上的平均置信度值。
    3. 具体描述：关注攻击成功的对抗样本在误分类下的平均置信度。
    4. 与鲁棒性关系：对抗分类的置信度也与模型的鲁棒性有关。在对抗分类上的平均置信度越高，说明此次攻击越成功，在相同条件下也说明模型的鲁棒性越脆弱。
    5. 指标计算公式：

    ACAC=1m∑i=1mPf(xiadv)(Aε,p(xi))ACAC=\ \frac{1}{m}\sum_{i=1}^{m}{P_{f(x_i^{adv})}(A_{\varepsilon,p}(x_i))} ACAC=​m​​1​​​i=1​∑​m​​P​f(x​i​adv​​)​​(A​ε,p​​(x​i​​))

    # ACTC

    1. 正确类别平均置信度。通过对对抗攻击样本（FGSM、PGD、C&W）的真实类，计算预测可信度的平均值。用来评估攻击在多大程度上偏离真实值。
    2. 物理含义：计算在对抗攻击成功的样本上的其真实分类的平均置信度。
    3. 具体描述：关注攻击成功的对抗样本在真实分类下的平均置信度。
    4. 与鲁棒性关系：同样，关注真实分类的置信度也与鲁棒性有关。误分类样本在真实分类上的置信度越高，说明此次攻击虽然发生了误分类，但程度不大。与ACAC结合来看作差值，ACAC-ACTC可以得出对抗攻击在多大程度上越过了决策边界。
    5. 指标计算公式：

    ACTC=1m∑i=1mPyi(Aε,p(xi))ACTC=\ \frac{1}{m}\sum_{i=1}^{m}{P_{y_i}(A_{\varepsilon,p}(x_i))} ACTC=​m​​1​​​i=1​∑​m​​P​y​i​​​​(A​ε,p​​(x​i​​))

    # ALDp

    1. 平均Lp失真度（使用I-FGSM，对于每个样本，直到模型产生误分类）。具体来说，计算扰动后发生改变的像素数量；计算原始示例和扰动示例之间的欧氏距离 ；测量对抗样本全维度下最大变化量。ALDp为所有攻击成功的对抗样本的平均归一化Lp失真度，ALDp越小，对抗样本的不可感知性越强。
    2. 物理含义：在不同范数下，计算测试集中的对抗样本与原始图片之间的扰动与图像的比值，衡量对抗样本的视觉可见性。
    3. 具体描述：在不同范数条件下，利用扰动大小之于原始数据的比值，来衡量对抗样本的视觉可见性。
    4. 指标计算公式：

    ALDp=1m∑i=1m∣∣xadv(i)−x(i)∣∣p∣∣x(i)∣∣p{ALD}_p=\frac{1}{m}\sum_{i=1}^{m}\frac{{||x_{adv}^{\left(i\right)}-x^{(i)}||}_p}{{||x^{(i)}||}_p} ALD​p​​=​m​​1​​​i=1​∑​m​​​∣∣x​(i)​​∣∣​p​​​​∣∣x​adv​(i)​​−x​(i)​​∣∣​p​​​​

    # ASS

    1. 平均结构相似性（使用I-FGSM，对于每个样本，直到模型产生误分类）。为了评估对抗样本的不可感知性，ASS被定义为所有攻击成功对抗样本与其原始样本间的平均相似性。
    2. SSIM更符合人类视觉感知，计算所有对抗样本与原始样本间的SSIM并取其平均值。
    3. 指标计算公式：

    ASS=1m∑i=1mSSIM(xadv(i),xi)ASS=\frac{1}{m}\sum_{i=1}^{m}SSIM(x_{adv}^{(i)},x^i) ASS=​m​​1​​​i=1​∑​m​​SSIM(x​adv​(i)​​,x​i​​)

    # PSD

    1. 扰动敏感距离。用于评测人类对扰动的感知能力。
    2. 物理含义：基于对比度遮掩理论提出的一种计算方式评估人类视觉对扰动的感知程度。
    3. 具体描述：对于每个对抗样本的每个像素，取像素周围的矩形区域计算其标准差的倒数，计算其平均值，以表示人眼对对抗样本的感知程度。
    4. 指标计算公式：

    PSD=1n∑i=1n∑j=1tδ(j)(i)Sen(R(x(j)(i)))PSD=\ \frac{1}{n}\sum_{i=1}^{n}\sum_{j=1}^{t}{\delta_{\left(j\right)}^{\left(i\right)}Sen(R(x_{(j)}^{(i)}))} PSD=​n​​1​​​i=1​∑​n​​​j=1​∑​t​​δ​(j)​(i)​​Sen(R(x​(j)​(i)​​))

    其中n为对抗样本数量，
    t为像素点个数，
    δ(j)(i)\delta_{\left(j\right)}^{\left(i\right)}δ​(j)​(i)​​指第i个对抗样本中的第j个像素，
    R(x(j)(i))R(x_{(j)}^{(i)})R(x​(j)​(i)​​)指第i个样本中的第j个像素周围的矩形区域，
    Sen()Sen()Sen()表示标准差函数的倒数。

    # NTE

    1. 物理含义：对抗分类与其他分类中置信度最高的差值。
    2. 具体描述：对抗样本保持误分类情况下的噪声容忍性度量。

    # RGB

    1. 高斯模糊前后分类准确率比值
    2. 对高斯模糊鲁棒性(Robustness to Gaussian Blur)。高斯模糊常被用于计算机视觉算法中的图像去噪。正常情况下，一个高鲁棒性的对抗样本，在高斯模糊后应保持其误分类效果。
    3. 指标计算公式：

    RGBUA=count(F(GB(Xia))≠yi)count(F(Xia))≠yiRGB_{UA}=\ \frac{count(F(GB(X_i^a))\neq y_i)}{count(F(X_i^a))\neq y_i} RGB​UA​​=​count(F(X​i​a​​))≠y​i​​​​count(F(GB(X​i​a​​))≠y​i​​)​​

    RGBTA=count(F(GB(Xia))≠yi+)count(F(Xia))≠yi+RGB_{TA}=\ \frac{count(F(GB(X_i^a))\neq y_i^+)}{count(F(X_i^a))\neq y_i^+} RGB​TA​​=​count(F(X​i​a​​))≠y​i​+​​​​count(F(GB(X​i​a​​))≠y​i​+​​)​​

    UA表示非定向攻击，TA表示定向攻击，GB函数表示高斯模糊处理。RGB结果越高，说明对抗样本鲁棒性越强。

    # RIC

    1. 图像压缩前后分类准确率比值
    2. 对图像压缩鲁棒性(Robustness to Image Compression)。图像压缩常被用于计算机视觉算法中的图像去噪。正常情况下，一个高鲁棒性的对抗样本，在图像压缩后应保持其误分类效果。
    3. 指标计算公式：

    RICUA=count(F(IC(Xia))≠yi)count(F(Xia))≠yiRIC_{UA}=\ \frac{count(F(IC(X_i^a))\neq y_i)}{count(F(X_i^a))\neq y_i} RIC​UA​​=​count(F(X​i​a​​))≠y​i​​​​count(F(IC(X​i​a​​))≠y​i​​)​​

    RICTA=count(F(IC(Xia))≠yi+)count(F(Xia))≠yi+RIC_{TA}=\ \frac{count(F(IC(X_i^a))\neq y_i^+)}{count(F(X_i^a))\neq y_i^+} RIC​TA​​=​count(F(X​i​a​​))≠y​i​+​​​​count(F(IC(X​i​a​​))≠y​i​+​​)​​

    UA表示非定向攻击，TA表示定向攻击，IC函数表示图像压缩处理。RIC结果越高，说明对抗样本鲁棒性越强。

    # MCE(Mean Corruption Error)

    1. 计算神经网络对自然噪音的平均error值。
    2. 具体描述：计算分类器f在每个严重级别下对每个损坏类型的错误率，其均值为平均损坏误差。
    3. 指标计算公式：

    CECf=∑s=1tEs,cfCE_C^f=\sum_{s=1}^{t}E_{s,c}^f CE​C​f​​=​s=1​∑​t​​E​s,c​f​​

    其中，f表示分类器，C表示损坏方式，t表示严重级别，E表示错误率。
    式子含义可以解释为，对于每种损坏方式C计算其错误率 = C中每种不同严重级别的图片放入模型中的错误率的累加和mCE即等于各种损坏方式CEcfCE_c^fCE​c​f​​的均值。

    # RMCE(relative Mean Corruption Error)

    1. 计算神经网络在自然噪音下，error与vanilla差值。
    2. 具体描述：将每个损坏方式与模型的原始错误率作差值。如果一个分类器能承受大部分损坏（损坏鲁棒性越强），那么mCE和原始数据的错误率之间的差值越小。
    3. 指标计算公式：

    RmCEcf=∑s=1tEs,cf−EcleanfRmCE_c^f=\sum_{s=1}^{t}E_{s,c}^f-E_{clean}^f RmCE​c​f​​=​s=1​∑​t​​E​s,c​f​​−E​clean​f​​

    # MFP(Mean Flip Probability)

    1. 计算神经网络对自然噪音序列的error。
    2. 具体描述：作者用mFR来表示Imagenet-P里的原始图片生成的损坏图片序列中的两个相邻图片的分类差异。
    3. 指标计算公式：

    FPcf=1q(n−1)∑j=2n1(f(xi(j)≠f(xi(j−1)))FP_c^f=\frac{1}{q(n-1)}\sum_{j=2}^{n}1(f(x_i^{(j)}\neq f(x_i^{(j-1)})) FP​c​f​​=​q(n−1)​​1​​​j=2​∑​n​​1(f(x​i​(j)​​≠f(x​i​(j−1)​​))

    其中FP指反转（前一张图片与后一张图片分类不同）概率，
    q指选取的序列数目（即q张图片对应某一损坏方式生成的q个序列）
    n指这个序列中含有的图片数量。
    然后作者定义FRcf=FPcf/FPcbase{FR}_c^f={FP}_c^f/{FP}_c^{base}FR​c​f​​=FP​c​f​​/FP​c​base​​,mFR是针对各种损坏方式计算其FRcf{FR}_c^fFR​c​f​​的均值。作者在原论文中定义的基线模型是ALEXNET。

    # CAV

    1. 分类准确度方差，模型防御前后准确度的差
    2. 具体描述：描述模型防御前后对干净数据的分类差异。我们希望防御策略可以在保持对原始样本的分类精度的基础上增加其鲁棒性。
    3. 指标计算公式：

    CAV=ACC(fd,D)−ACC(f,D)CAV=ACC(f^d,D)-ACC(f,D) CAV=ACC(f​d​​,D)−ACC(f,D)

    其中fdf^df​d​​指应用了防御策略后的模型。

    # CRR/CSR

    1. 模型防御后，减少的错误分类百分比；模型防御后，增加的错误分类百分比。
    2. CRR指之前被模型错误分类，但在应用防御策略后被模型正确分类的样本比例。CSR指之前被模型正确分类，但在应用防御策略后被模型错误分类的样本比例。
    3. 与鲁棒性关系：从逻辑上来讲，一个有效的防御策略在应用后应该使CRR远大于CSR。
    4. 指标计算公式：

    CRR=1n∑i=1ncount((f(xi)≠yi)&(fd(xi)=yi))CRR=\frac{1}{n}\sum_{i=1}^{n}count((f(x_i)\neq y_i)\&(f^d(x_i)=y_i)) CRR=​n​​1​​​i=1​∑​n​​count((f(x​i​​)≠y​i​​)&(f​d​​(x​i​​)=y​i​​))

    1. 测量防御造成的置信方差（对防御前后均分类正确样本做统计）
    2. 具体描述：有一些样本可能在防御前后都被正确分类，但其置信度可能会发生变化。这一指标关注这部分样本的置信度变化。

    # ENI

    1. 使用Lipsch itz连续常数衡量模型对噪音的敏感性
    2. 物理含义：从李普希兹常数的角度出发，将损失函数视作一个以输入xix_ix​i​​为因变量的函数，计算其李普希兹常数。李普希兹常数越小，说明函数曲线越平滑，说明模型对于扰动越不敏感。
    3. 具体描述：基于这样的思想:如果两个样本“相似”，那么它们的测试误差就会非常接近。受此启发，作者从李普希茨常数的角度提出ϵ\epsilonϵ下的噪声不敏感性来衡量模型对广义噪声的鲁棒性。显然，较低的值表示更强的模型。首先随机选择N个原始样本，然后通过各种方法(例如，对抗攻击、高斯噪声、模糊等)从每个干净的例子中生成M个样本。当一个干净的例子和相应的污染例子被输入模型时，计算模型损失函数之间的差异。损失函数中不同的严重程度被用来衡量模型对约束ϵ\epsilonϵ内的广义小噪声的不敏感性和稳定性。
    4. 与鲁棒性的关系：这项指标关注在ϵ\epsilonϵ限定下的对抗样本与原始样本的损失函数差异，平均差异越小，说明模型对小于ϵ\epsilonϵ的噪声越不敏感，鲁棒性越强。

    # BD

    1. 随机寻找N个正交的方向，然后计算对于一个模型来说每个方向需要移动多少可以改变instance的prediction label，来衡量模型的决策边界最小距离。
    2. 物理含义：使用EBD算法来计算每个数据点的对抗距离上边界，求其均值。
    3. 具体描述：EBD本质上使用论文中提出的攻击算法，近似得到扰动最小的、使模型发生误分类的上边界距离。EBD的计算方式是随机定义m（m<=数据维数）个正交的向量，对于每个方向，每移动一定的均方根距离就判断一次是否发生误分类，如果发生误分类那这个距离就是这个方向上的对抗上边界距离。找到m个方向上的最小值，近似为对抗上边界距离。计算每个样本点的上边界距离均值。
    4. 与鲁棒性的关系：EBD距离越大，说明模型鲁棒性越强。

    1. 神经元边界覆盖
    2. 具体描述：训练集中的样本在神经元上的输出不会出现在极端案例区，如果测试集与训练集是同分布的，那么测试集中的样本也几乎不会出现在极端案例区。但神经元的极端区域覆盖率也很重要，因为模型的脆弱性也可能表现在极端案例区。
